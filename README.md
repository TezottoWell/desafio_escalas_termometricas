# Desafio Escalas Termométricas

Este projeto é a solução para um desafio proposto durante o curso "Go Developer" da plataforma DIO. O objetivo principal é implementar funcionalidades relacionadas à conversão e manipulação de escalas termométricas.

## Como Executar

1.  **Clone o repositório:**
    ```bash
    git clone https://github.com/TezottoWell/desafio_escalas_termometricas
    cd desafio_escalas_termometricas
    ```

2.  **Execute a aplicação:**
    ```bash
    go run app.go
    ```
